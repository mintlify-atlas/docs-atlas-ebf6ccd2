## Welcome to Tendon

Tendon is a comprehensive Next.js template that combines the best modern web technologies to give you a head start on building production-ready applications. With authentication, database integration, and a complete UI component library pre-configured, you can focus on building features instead of setting up infrastructure.

## Key Features

- **Modern Authentication** - Complete auth system powered by Better Auth with email/password support
- **Type-Safe Database** - Drizzle ORM with full TypeScript support and migrations
- **Edge-Ready Database** - Turso/libSQL for local development and global edge deployment
- **Beautiful UI** - 50+ pre-built components using Radix UI primitives
- **Dark Mode** - Built-in theme support with next-themes
- **Form Handling** - React Hook Form with Zod validation
- **Modern Tooling** - Vite-powered Next.js with Vinext for faster builds
- **Production Ready** - Configured for deployment to Vercel with environment variables

## What's Included

### Core Framework
- **Next.js 16** - Latest React framework with App Router
- **React 19** - Latest React with server components
- **TypeScript** - Full type safety across the stack
- **Vinext** - Vite-powered Next.js for blazing fast development

### Authentication & Database
- **Better Auth** - Modern authentication library with session management
- **Drizzle ORM** - Type-safe SQL query builder
- **Turso/libSQL** - SQLite for local dev, distributed database for production

### UI & Styling
- **Tailwind CSS 4** - Utility-first CSS framework
- **Radix UI** - Unstyled, accessible component primitives
- **Lucide Icons** - Beautiful, consistent icon library
- **Framer Motion** - Smooth animations and transitions
- **next-themes** - Dark mode support

### Components & Forms
- **React Hook Form** - Performant form handling
- **Zod** - TypeScript-first schema validation
- **Sonner** - Toast notifications
- **CMDK** - Command palette component
- **Recharts** - Composable charting library

### Developer Experience
- **ESLint** - Code linting and formatting
- **Drizzle Kit** - Database migrations and introspection
- **shadcn** - Component CLI for adding UI components
